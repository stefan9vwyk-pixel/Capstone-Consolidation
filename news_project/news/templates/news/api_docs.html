GET POST /api/articles/

Manage and query application stories. Readers read approved text fields; authors submit entries to editorial lines.

Query Parameters

  • approved (boolean) — Filter entries by current database approval state.
  • publisher (int) — Filter content rows by a specific unique publisher ID.
  • q or search — Full text query matching against titles or content bodies.
GET POST /api/newsletters/

Retrieve, manage, or compose active newsletter volumes distributed by active publishing nodes.

EVENT Webhook System Signal Trigger

System triggers emit automated POST records tracking payload states whenever an entry completes validation checks.
